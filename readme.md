# 💬 Chatbot com Python e Streamlit

Projeto de chatbot desenvolvido em Python utilizando Streamlit para interface web e integração com modelo de Inteligência Artificial.

> 📌 Este projeto foi desenvolvido como parte da **Jornada Python da Hashtag Treinamentos**, com o objetivo de aplicar na prática conceitos de construção de interfaces e integração com IA.
> 🚧 Projeto em desenvolvimento — a integração completa com a API de IA depende da configuração de uma chave de acesso.

## 🚀 Tecnologias

* Python
* Streamlit
* OpenAI API

## ⚙️ Funcionalidades

* Interface de chat interativa no navegador
* Armazenamento do histórico de mensagens
* Envio de mensagens do usuário
* Estrutura pronta para integração com IA

## ▶️ Como executar

1. Instale as dependências:

```bash
pip install streamlit openai
```

2. Execute o projeto:

```bash
streamlit run main.py
```

## 📊 Funcionamento

* O usuário envia mensagens através da interface web
* As mensagens são armazenadas durante a sessão
* O sistema está preparado para enviar essas mensagens a um modelo de IA e exibir as respostas

## ⚠️ Observações

* É necessário configurar uma **API Key da OpenAI** para ativar as respostas da IA

## 📚 Objetivo

Praticar o desenvolvimento de aplicações interativas, manipulação de estado em aplicações web e integração com APIs de Inteligência Artificial.

## 🧠 Aprendizados

* Criação de interfaces web com Streamlit
* Gerenciamento de estado com `session_state`
* Estrutura de um chatbot moderno
* Integração com APIs externas
